DESCRIPTION: 
Campsite commander is an app that gathers information about a trip planned by families. this app helps to calculate the total capacity of lugages , the catergories how much gear is taken and the name of the items 



FEATURES:
Item name: name of items being taken with 
Category of item: sums up the item and where it belongs 
Quantity: total number of items 
add gear button: adds all the gear together 

Home screen:
Beginning point of the app.
Displays app title and welcome message.
displays "campsite commander"

Main screen:
Displays "add gear" button
displays total items packed 

Score screen:
Displays final score.
Optional message based on performance (e.g., "Great job!" or "Try again!").
"Review"Button for reviewing the questions answered.

Detailed view screen:
Displays the full list of gear with details
"Back base button" to return to main screen
